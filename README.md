物理学生来到了物理学世界，在这里物理学家指引着他寻求真理，并赐予了他圣物，以帮助他打败混沌愚昧之怪。闯过重重关卡之后，学生以为会寻得真理，却发现将要击败的是这些他景仰的物理学家们。 游戏中，玩家将操控“物理学生”以及三个圣物，包括引力，时间和温度。每种圣物有两种相对的模式，玩家需要巧妙切换模式和运用圣物来通关。
